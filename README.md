# HR Analytics Dashboard 📊

An interactive dashboard analyzing employee attrition to identify turnover drivers and improve retention.

## 🚀 Key KPIs
* *Total Employees:* 1,470
* *Attrition Count:* 237 (Rate: 16.1%)
* *Avg Age:* 37 | *Avg Salary:* $6.5K | *Avg Tenure:* 7 yrs

## 🔍 Key Insights
* *High-Risk Age:* 26-35 group (116 departures).
* *Education:* Life Sciences (38%) & Medical (27%) are top attrition areas.
* *Salary Impact:* 163 employees left in the "Upto 5k" bracket.
* *Roles:* Laboratory Technicians & Sales Executives show the highest turnover.

## 🛠️ Tech Stack
* *Tool:* Power BI / Tableau
* *Skills:* Data Modeling, DAX, Data Visualization

## 💡 Recommendations
1. *Salary Review:* Target the "Upto 5k" bracket for retention incentives.
2. *Career Pathing:* Focus on the 26-35 age group for internal growth.
3. *Onboarding:* Address the spike in attrition seen in the first 2 years.
